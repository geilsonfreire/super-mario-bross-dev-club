# 🎮 Super Mario Bros - Dev Club 

- Deploy: https://geilsonfreire.github.io/super-mario-bross-dev-club/

Bem-vindo ao **Super Mario Bros Dev Club**! Este é um projeto desenvolvido pela comunidade de desenvolvedores focado em criar uma experiência interativa do clássico jogo Super Mario Bros utilizando **HTML5**, **CSS3** e **JavaScript**.

<img src="./img/Ap-Sotware-Design.gif" alt="Logo do Super Mario Bross">

## 📋 Sobre o Projeto

Este projeto é uma recriação educacional do icônico jogo Super Mario Bros, desenvolvido como parte de um clube de desenvolvimento. O objetivo é aprender e praticar conceitos fundamentais de desenvolvimento web enquanto nos divertimos com um clássico personagem do SUPER MARIO BROSS.

## 🚀 Funcionalidades

- 🎨 Arte e sprites em estilo retro
- 🔊 Suporte a vídeos e mídias
- 📱 Interface responsiva

## 📁 Estrutura do Projeto

```
super-mario-bross-dev-club/
├── index.html          # Arquivo HTML principal
├── style.css           # Estilos CSS
├── script.js           # Lógica JavaScript
├── README.md           # Este arquivo
├── img/                # Imagens e sprites
├── public/             # Arquivos públicos
└── video/              # Vídeos do projeto
```

## 💻 Como Usar

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Nenhuma dependência externa necessária

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/geilsonfreire/super-mario-bross-dev-club.git
```

2. Abra o arquivo `index.html` no seu navegador
```bash
# No Windows
start index.html

# No macOS
open index.html

# No Linux
xdg-open index.html
```

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura da aplicação
- **CSS3** - Estilização e animações
- **JavaScript (Vanilla)** - Lógica do jogo e interatividade

## 📚 Conceitos Aprendidos

Este projeto aborda os seguintes conceitos de desenvolvimento web:

- Manipulação do DOM
- Detecção de eventos de teclado
- Animações CSS
- Canvas API (se aplicável)
- Programação orientada a objetos
- Controle de física básica (gravidade, colisão)
